# FUTURE_DS_02

# Task 2: Social Media Campaign Performance Tracker

## Overview
This project is part of the **Data Science & Analytics Internship** at **Future Interns**.
The objective of this task is to analyze social media ad campaign data and evaluate
campaign performance using key marketing metrics.

The dashboard helps understand how different campaigns and platforms perform
in terms of spend, engagement, and conversions.

---

## Dataset
- Simulated Facebook & Instagram ad campaign data
- Includes impressions, clicks, spend, and conversions
- Used for performance comparison and ROI analysis

---

## Tools & Technologies
- Power BI Desktop
- Microsoft Excel
- DAX (for KPI calculations)

---

## Key KPIs Analyzed
- Total Spend
- Total Impressions
- Total Clicks
- Click Through Rate (CTR)
- Return on Investment (ROI)
- Conversions

---

## Dashboard Highlights
- Campaign-wise Spend and CTR comparison
- Clicks vs Spend analysis using bubble chart
- Platform performance comparison (Facebook vs Instagram)
- ROI breakdown using conversion insights
- Interactive slicers for Campaign, Age, and Gender

---

## Key Insights
- Higher spend does not always guarantee higher CTR
- Instagram shows stronger engagement in certain campaigns
- Campaign efficiency varies significantly across platforms
- Clear relationship observed between spend and clicks

---

## Deliverable
An interactive Power BI dashboard designed to help marketing teams
track campaign performance and make data-driven decisions.
